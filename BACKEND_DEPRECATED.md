# Backend Acknowledgement

The backend services for Paulie's Prediction Partners are **currently deprecated**. All frontend code (web pages, desktop apps, UI assets) has been removed from this repository. No further work should be done on the backend unless or until a concrete user request is received to support a new frontend implementation.

## Guidelines

* Treat the backend as frozen: no new features, no bug fixes, and no refactoring unless instructed.
* When a request comes in to update or replace the frontend, the backend may be reviewed and refactored at that time to support the new design.
* Screenshots and design assets have been preserved elsewhere; they are not part of this repo.

This document serves as the canonical reminder that development focus has shifted away from the existing backend until the front end is rebuilt.
